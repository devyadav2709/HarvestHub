# Harvest Hub

Harvest Hub is a web-based agricultural marketplace developed using PHP and MySQL. The platform connects farmers and consumers, allowing farmers to sell agricultural products directly while consumers can browse, bid, and purchase products online.

## Features

### 👨‍🌾 Farmer Features
- Farmer Registration and Login
- Add and Manage Products
- Upload Product Images
- View and Accept Bids
- Farmer Dashboard

### 🛒 Consumer Features
- User Registration and Login
- Browse Agricultural Products
- Place Bids on Products
- Purchase Products
- View Purchase History

### 📊 Additional Features
- Crop Price Information
- Government Subsidy Details
- Farming Equipment Information
- Contact and Feedback System
- Admin Panel
- Responsive Design

## 🛠️ Technologies Used

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/devyadav2709/HarvestHub.git
```

2. Move the project folder into XAMPP htdocs.

3. Create a MySQL database.

4. Import the SQL file from the `sql` folder.

5. Configure database credentials.

6. Start Apache and MySQL.

7. Open in your browser:

```text
http://localhost/HarvestHub
```
